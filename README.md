# About the repository
This repository contains some very simple scripts supporting an introductory course on Reliability, Availability, Maintainability and Supportability (RAMS). Here is a survey of the scripts:
- [Fitting a Poisson distribution from arrivals](https://nbviewer.jupyter.org/github/chrisrijsdijk/RAMS/blob/master/sitetext/PoissonDistribution.ipynb?flush_cache=true).
- [Fitting a Weibull distribution from life times](https://nbviewer.jupyter.org/github/chrisrijsdijk/RAMS/blob/master/sitetext/WeibullFitting.ipynb?flush_cache=true).
- [Reliability and availability assessments using a state transition matrix](https://nbviewer.jupyter.org/github/chrisrijsdijk/RAMS/blob/master/sitetext/StateTransitionMatrix.ipynb?flush_cache=true).
- [Supervised and unsupervised machine learning  applied to oilanalysis](https://nbviewer.jupyter.org/github/chrisrijsdijk/RAMS/blob/master/sitetext/Oilanalysis.ipynb?flush_cache=true).
- [Causal inference applied to a Fault Detection and Diagnostics case](https://nbviewer.jupyter.org/github/chrisrijsdijk/RAMS/blob/master/sitetext/CausalInference.ipynb?flush_cache=true).
